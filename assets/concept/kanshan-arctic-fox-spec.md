# 刘看山北极狐角色规范

刘看山应定义为白色北极狐 IP。核心形象来自 `source-image` 目录中的样图：白色块面身体、黑色粗线外轮廓、长而平的鼻吻、黑色椭圆鼻头、点状眼睛、尖耳、小圆尾和线状四肢。角色可以制作成 3D toon 资产。toon 指让 3D 模型呈现卡通平面感的渲染方式。

## 核心识别点

| 元素 | 规范 |
| --- | --- |
| 物种 | 北极狐。识别重点是长鼻吻、尖耳、白色身体和尾部。不要按普通狗、狼或圆球宠物处理。 |
| 整体轮廓 | 头部和躯干是一体化白色大形。外轮廓接近竖向圆角方块或梨形，侧面可被长鼻吻拉长。 |
| 身材 | 轻盈、简洁、略圆润。不能做成写实狐狸的细长躯干，也不能做成无鼻吻的胖球。 |
| 头部 | 头部不单独分块。额头、背部和躯干用连续轮廓连接，保留简笔画低细节感。 |
| 鼻吻 | 最重要的识别点。侧面鼻吻长而平，像从脸部水平伸出的白色长方形或楔形。正面可只露出短鼻梁和小鼻头。 |
| 鼻头 | 黑色、圆角、体积明显，位于鼻吻最前端。侧面近似大椭圆或圆角胶囊，正面近似小椭圆。 |
| 眼睛 | 黑色点状眼睛。默认只用圆点或短弧线表达，不增加眼白、虹膜和复杂眉眼。 |
| 耳朵 | 两个尖耳从头顶轮廓长出。尺寸克制，轮廓像小三角形。不要做成兔耳、大狗耳或写实狐耳。 |
| 尾巴 | 样图中尾巴多为身体后侧的小圆形或短弯尾。3D 化时可略增加体积，但应保持轻巧，不做成大蓬尾。 |
| 四肢 | 手脚是黑色线状结构。手臂细、可弯曲，末端可做三指小手。腿细短，脚底可有小折角。 |
| 表情 | 低复杂度。优先通过点眼、短弧眼、头部角度、手势和身体姿态表达情绪。 |
| 描边 | 黑色粗描边是主要风格特征。Three.js 中可通过 toon 材质、反向法线描边或屏幕后处理实现。 |
| 色彩 | 主体为白色或微暖白。鼻头、眼睛和线状四肢为黑色。道具和情绪符号可使用少量高饱和色。 |

## 三视图要求

| 视图 | 必须表达 |
| --- | --- |
| 正面 | 一体化白色身体、两只尖耳、两只点眼、小黑鼻、线状手脚。尾巴可从身体一侧露出为小圆形或短弯形。 |
| 侧面 | 长鼻吻、黑色大鼻头、单只点眼、头身一体轮廓、细线手脚、小圆尾或短弯尾。鼻吻长度必须明显。 |
| 背面 | 不显示鼻吻和鼻头。重点显示双尖耳、白色背部大形、后侧小尾巴、两条细腿和手臂挂点。 |

## 比例要求

- 标准站姿高度记为 1.0。
- 正面身体宽度建议为高度的 0.52 到 0.66。
- 侧面鼻吻从脸部伸出的长度建议为身体高度的 0.28 到 0.40。
- 黑色鼻头长度建议为鼻吻长度的 0.28 到 0.38。
- 腿长建议为高度的 0.18 到 0.26。
- 线状手臂直径建议明显细于身体外轮廓描边，但在 3D 中要有可绑定骨骼的厚度。
- 尾巴以小圆尾或短弯尾为默认方案，外露宽度建议为身体宽度的 0.12 到 0.22。
- 黑色外轮廓要厚、干净、稳定，避免细碎毛发和复杂明暗。

## 3D 转译要求

| 项目 | 要求 |
| --- | --- |
| 主体网格 | 使用简洁的白色一体网格承载头部和身体。不要把头、脖子和躯干切成写实动物结构。 |
| 鼻吻结构 | 鼻吻需要独立可控或有清晰边线，便于侧面识别和轻微动画。 |
| 鼻头结构 | 鼻头可作为黑色独立网格或材质区域。侧面必须保持圆润体积。 |
| 四肢结构 | 四肢可以比样图线条略有厚度，便于蒙皮和挂点。视觉上仍应读作黑色线条。 |
| 尾巴结构 | 尾巴使用小圆尾或短弯尾。若需要 3D 体积，保持贴近身体，不抢鼻吻识别。 |
| 材质 | 主体使用白色 toon 材质。鼻头、眼睛和四肢使用黑色 toon 材质。避免写实毛发、皮革光泽和复杂反射。 |

## 骨骼要求

| 骨骼 | 用途 |
| --- | --- |
| `root` | 角色整体移动和缩放。 |
| `body` | 身体呼吸、重心和整体姿态。 |
| `head` | 点头、转头和视线方向。头身一体时，该骨骼控制上半身朝向。 |
| `snout` | 鼻吻方向和轻微摆动。用于强调好奇、嗅闻和拖拽反馈。 |
| `tail` | 小尾巴摆动、紧张收起和开心翘起。 |
| `arm-left` | 左手动作和道具挂载。 |
| `arm-right` | 右手动作和道具挂载。 |
| `leg-left` | 左脚走路、跑步和跳跃。 |
| `leg-right` | 右脚走路、跑步和跳跃。 |

## 第一批动作

| 动作 | 要求 |
| --- | --- |
| `idle` | 轻微呼吸，点眼保持安静，小尾巴低幅摆动。 |
| `walk` | 轻快小步，细腿有清楚落点，身体不要大幅左右摇晃。 |
| `run` | 更高频步伐，可加入身体前倾和速度线，但不破坏一体化轮廓。 |
| `blink` | 点眼短暂变成短弧或消失一帧，保持低复杂度。 |
| `happy` | 可跳起、挥手或比心。尾巴上扬，点眼可变短弧。 |
| `dragging` | 鼻吻或身体沿拖动方向产生轻微拉伸，四肢有滞后反馈。 |
| `hungry` | 身体轻微前倾，手靠近腹部，可挂口水或食物图标。 |
| `sleepy` | 头部下沉，眼睛变短弧，尾巴和手臂下垂。 |
| `sick` | 身体轻颤，点眼无神，可挂汗滴、阴云或冰袋特效。 |

## 挂点要求

| 挂点 | 用途 |
| --- | --- |
| `head` | 帽子、光环、冰袋。 |
| `mouth` | 鼻吻下方的叼取物、气泡、口水。 |
| `hand-left` | 左手道具。 |
| `hand-right` | 右手道具。 |
| `tail` | 小尾巴装饰或尾巴特效。 |
| `feet` | 滑板、地面阴影、落地特效。 |
| `emotion` | 爱心、音符、汗滴、阴云。 |
